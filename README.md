📊 Previsão de Score de Crédito com Machine Learning
Projeto desenvolvido durante a Jornada Python da Hashtag Treinamentos. O objetivo foi criar modelos de classificação para prever o score de crédito de clientes e identificar os fatores que mais
influenciam essa pontuação.

🎯 Resultados
Modelos Treinados: Árvore de Decisão (Random Forest) e KNN (K-Nearest Neighbors).

Performance: Alcançamos uma acurácia de 82% no melhor modelo.

Insights: Identificação das características mais relevantes para a definição do perfil do cliente.

🛠️ Etapas do Projeto

Importação: Leitura da base de dados original. 📥

Limpeza: Tratamento de valores vazios e dados inconsistentes. 🧹

Encoding: Transformação de textos em números para que os modelos possam aprender. 🔢

Treinamento: Aplicação dos algoritmos de Machine Learning. 🤖

Avaliação: Comparação de performance para escolher o modelo mais preciso. ✅

Análise de Importância: Verificação de quais colunas mais impactam o score. 📈
